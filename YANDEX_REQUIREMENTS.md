# Требования Яндекс Игр

Дата обновления: 16 декабря 2025 года

## Ключевые технические требования

### 1. SDK и интеграция
- [x] **1.1** SDK Яндекс Игр встроен
- [x] **1.2** Не требуется регистрация на сторонних сервисах
- [x] **1.3** При сворачивании страницы звук останавливается
- [x] **1.5** Реклама только через SDK
- [x] **1.12** Подключена монетизация РСЯ

### 2. Мобильные устройства (1.6.1)
- [x] **1.6.1.1** Полноэкранный режим
- [x] **1.6.1.3** Элементы не деформируются при изменении ориентации
- [x] **1.6.1.5** Управление жестами (джойстики)
- [x] **1.6.1.8** Лонгтап не вызывает контекстное меню

### 3. Десктоп (1.6.2)
- [x] **1.6.2.1** Активное поле растягивается до края
- [x] **1.6.2.3** Элементы не деформируются
- [x] **1.6.2.4** Управление клавиатурой/мышью
- [x] **1.6.2.7** Нет контекстного меню при клике

### 4. UI и отображение
- [x] **1.8** Размеры кнопок адаптированы (минимум 44px)
- [x] **1.10.1** Масштаб не выходит за рамки экрана
- [x] **1.10.2** Нет браузерной прокрутки
- [x] **1.10.3** Элементы не накладываются друг на друга
- [x] **1.10.4** Управление одной рукой возможно

### 5. SDK методы (1.19)
- [x] **1.19.1** SDK инициализируется правильно
- [x] **1.19.2** Вызывается `LoadingAPI.ready()` когда игра готова
- [ ] **1.19.3** GameplayAPI.start()/stop() - опционально
- [ ] **1.19.4** game_api_pause/resume события - опционально

### 6. Сохранения (1.9)
- [x] Прогресс сохраняется после действия игрока
- [x] Обновление страницы не влияет на сохранения
- [x] Используются облачные сохранения (указать в черновике)

## Чеклист перед публикацией

### Техническое
- [ ] Консоль без ошибок
- [ ] Размер < 100 МБ
- [ ] index.html в корне архива
- [ ] Нет пробелов/русских символов в именах файлов

### SDK
- [ ] `YaGames.init()` вызывается корректно
- [ ] `ysdk.features.LoadingAPI.ready()` вызывается
- [ ] Реклама через `ysdk.adv.showFullscreenAdv()`
- [ ] Rewarded через `ysdk.adv.showRewardedVideo()`
- [ ] Лидерборд через `ysdk.getLeaderboards()`
- [ ] Сохранения через `ysdk.getPlayer().then(p => p.setData())`

### UI
- [ ] Кнопки минимум 44x44px
- [ ] Нет перекрытий элементов
- [ ] Работает в landscape и portrait
- [ ] Safe area для notch устройств

### Контент
- [ ] Нет внешних ссылок
- [ ] Нет политики/религии
- [ ] Нет эзотерики
- [ ] Авторские права соблюдены

## Частые причины отклонения

1. **Звук не останавливается** при сворачивании (1.3)
2. **Элементы перекрываются** на разных разрешениях (1.10.3)
3. **Мелкие кнопки** на мобильных (1.8)
4. **Нет LoadingAPI.ready()** (1.19.2)
5. **Браузерный скролл** появляется (1.10.2)
6. **Контекстное меню** при лонгтапе (1.6.1.8)
